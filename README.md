**ReAL-Health 2026: Reliable AI in Healthcare**

A workshop in the NeurIPS 2026 program.
Web URL: https://real-health-neurips.github.io/
Workshop: December 2026, NeurIPS 2026 (Atlanta, Paris, or Sydney; venue TBA after acceptance notification)

This site is built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme.

Local preview:
```
bundle install
bundle exec jekyll serve
```
Then open http://localhost:4000

To deploy on GitHub Pages, create a GitHub organization named `real-health-neurips` (or any name matching the URL you want), put these files in a repository named `<orgname>.github.io`, push, and enable Pages (Settings, Pages, deploy from main, root). Replace `YOUR_USERNAME` in `_config.yml` with your GitHub username after deployment if you want a working footer social link.

Items marked **TBA** in the pages should be updated as the program is finalized:
- Workshop URL (in proposal and home)
- Talk titles and abstracts for invited speakers
- OpenReview submission link
- Specific workshop day within December 2026
- Accepted-papers list (after Sept 29, 2026 notification)

Placeholder portraits (`assets/img/people/*.svg`) can be replaced with real photos at the same paths.
